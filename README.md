# ecommerce-front-end

## Description
An e-commerce front-end built with React, TypeScript, and Vite. The app provides product browsing, product details, cart management, checkout flow, order confirmation, and an admin area for managing products and orders.

## Dependencies
### Runtime
- React
- React DOM
- React Router DOM
- React Icons

### Development
- Vite
- TypeScript
- ESLint
- @vitejs/plugin-react
- @types/node
- @types/react
- @types/react-dom
- @types/react-router-dom
- eslint-plugin-react-hooks
- eslint-plugin-react-refresh
- globals
- typescript-eslint

## Setup
1. Install dependencies:
   - `npm install`
2. Start the development server:
   - `npm run dev`
3. Build for production:
   - `npm run build`
4. Preview the production build:
   - `npm run preview`
5. Run linting:
   - `npm run lint`
